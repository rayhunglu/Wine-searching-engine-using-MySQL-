About us

OUR STORIES


This is a project of cs 540. The main goal of our project is finding the recommended wines according to users’ preference based on different search constraints. To start, the most important thing is getting some good and reliable databases. Then，we use ONID Data Server from OSU as our server for SQL. Next thing we did is building the environment for our website. In the list of recommended wines, there are buttons to link to the detailed description of wine. The extra new page with description lets users know more about their fancy wine.

Why choose us

FIRST

We found two data sets on Kaggle that one has 150k rows of wine reviews while the other has 130k rows.

MORE
SECOND

we searched for the data set of top 100 wines to further improve the ability for suggesting wines.

MORE
THIRD

We chose the attributes of ranking, winery, the name of the wine, score, and description to match our previous data sets.
